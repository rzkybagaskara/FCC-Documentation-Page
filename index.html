Introduction

Python originally developed for Amoeba Operating System as high-scripting language based on C Programming Language. Python was created by Guido Von Rossum in Amsterdam, Netherlands back in 1990. The reason why Guido named his programming language "Python" is because he was a huge fan of a show called "Monty Python". Therefore, he chose the name "Python" as it's name. From it's release in 1990, Python popularity has grow skyrocket. The latest version of Python is Python version 3.

Python have two types of compiling mods, they are:

  • Interactive

    Based of interpreter. Each code will be executed per line.

  • Script

    The code will be executed if we click Run on it. Also, this type of compiler is affected by indentation. So, be careful with it.

Nowadays, Python can be use for anything. From web browsing to high tech stuff such as A.I, image recognition, autonomous vehicle, etc. Python also one of the programming languages that user-friendly, especially to the beginners.

Rules

Because Python is based on C Programming Language, there are some similarities and differences between those two languages, they are:

  • There is no declaration, including data types, argumentation, and variables.
  • Each subprograms are identify between indentation, so indendation plays a huge part in Python.

if...else statements
if statements are use for decision-making operations. There are 4 types of if statements in Python, including:
  1. if

    if is use for one condiiton. If the condition match the statement (which is True), then the result will be pass on to the next statement below. The basic form of if:
    if (condition): statement
    For example:

    a = 33
    b = 200
    if b > a:     print("b is greater than a")

    In this example we use two variables, a and b, which are used as part of the if statement to test whether b is greater than a. As a is 33, and b is 200, we know that 200 is greater than 33, and so we print to screen that "b is greater than a

  2. if...else

    if...else are use for two condition. If the first condition match the decision (which is True), then the condition will continue to the True statement below, whereas the False one will be taken into the False statement (which is else). The basic form of if...else:
    if (condition1):
    statement1 else: statement2
    For example:

    a = 3
    if (a % 2 == 0):
    print(' a is even') else:
    print('a is odd')

    The output will be "a is odd", because 3 divided by 2 is not zero (it still has remainder). Therefore, the condition will be in False statement which is "a is odd".

  3. if...elif...else

    if...elif...else are use for condition that has many outputs (more than two). The basic form of if...elif...else:
    if (condition1): statement1 elif (condition2): statement2 elif (condition3): statement3 else: statement4
    For example:

    score = int(input('Please input your score: '))
    if score >= 90:
    print('You got A') elif score <=85: print('You got B') elif score <= 80: print('You got C') else:
    print('You got D')

    If the input score is 90, then the output will be "You got A", so on and so forth.

  4. Nested if

    Basically, nested if are if inside of another if. There will be another selection inside of those if. The basic form of Nested if:
    if(condition1):
    if(condition2): statement else: statement

Looping Structure
Loops are being used to execute a block of program until certain amount or condition, depend on the situation. In Python, there are two types of Loops, such as:
  1. for

    A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string) that already being set the maximum amount of loop.
    for (variable) in (object):
    statement else:
    statement
    For example:

    for i in range (0,9): if i % 3 == 0:
    print(i)

    Output:
    0
    3
    6

  2. while

    With the while loop we can execute a set of statements as long as a condition is true.
    while (condition):
    statement     For example:

    a = 0
    b = 5
    while a < b: a += 1
    print(a)

    Output:
    1
    2
    3
    4
    5

Array
Array are used to store multiple values of data (homogen) in a single variable. In Python, Array are called "List". In Python, List can be use with brackets [ ]. The basic structure of List in Python are:
array_name = [list_of_array]

For example:


mobil = ['Honda','Ford','Mazda']
print(mobil)

Output:
['Honda'. 'Ford'. 'Mazda']
